Slider attributes:

  • data-arrows="true" : create slider control with prev and next arrows(this attribute has a default "true");
  • data-carousel="true" : creates a scrolling slider in a circle(this attribute has a default "true");
  • data-dotted="true" : create slider control with dotted paginator;
  • data-pagination="true" : create display position of current slide;
  • data-cells="true" : it includes the contents of the slides in the specified quantity;
  • data-animate="fade" : animates the change slide with fade;
  • data-touch="true" : create change slides with swipes;
  • data-touch="true" : create autoslide function in ms;

Examples:

Custom slider without attributes


	<div class="slider-wrapper"
		<div class="slide">...</div>
		...
	</div>

Slider with dottes and pagination


	<div class="slider-wrapper" data-arrows="false" data-pagination="true" data-dotted="true"
		<div class="slide">...</div>
		...
	</div>

Slider with 3 items in slide and pagination


	<div class="slider-wrapper" data-arrows="true" data-cells="3" data-pagination="true"
		<div class="cell-1">...</div>
		...
	</div>

Slider with dottes and auto change slide every 3 seconds 


	<div class="slider-wrapper" data-arrows="false" data-dotted="true" data-autoslide="3000"
		<div class="slide">...</div>
		...
	</div>
